# 本地 APK 可视化打包器

双击本目录的 `start-fixed-app-builder.bat`，或者在项目根目录双击 `build-apk-gui.bat`。

这个工具会做这些事：

- 输入服务器 IP/域名和网页端口。
- 可选择登录模式：自动检测、强制后台登录、强制免登录。
- 可填写购买卡密链接兜底地址；后台已配置购买链接时，后台配置优先生效。
- 使用 `tools` 目录内置的 JDK、Android SDK、Gradle 在电脑本机打包 APK。
- 打包时把后台地址写入 APK 内部的 `radar-app-config.json`。
- 生成的 APK 使用自动登录模式：检测到有效后台 API 就走后台登录；没有后台、只有前端页面时免登录进入主页。
- APK 默认输出到桌面 `ALinRadar-APK` 文件夹，也可以在界面里自己选择目录。

当前服务器推荐填写：

```text
协议：http
服务器IP/域名：192.140.166.49
网页端口：85
```

当前服务器公网访问只用 `85` 端口，打包 APK 时也填写 `85`。

如果只填前端地址、没有 PHP 后台，登录模式选“自动检测（推荐）”即可，打包出来的 APK 会免登录进入主页。如果确定这个地址有后台，可以选“强制后台登录”。
