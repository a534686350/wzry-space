## 已确认的协议事实

- `home-server-0.0.1-SNAPSHOT.jar` 只是保存和转发游戏数据，不解释英雄/技能字段。
- 共享端写入格式：`gameData[==]房间号[==]payload`。
- 网页/APK 拉取格式：`web时间戳[==]房间号`。
- WebSocket 返回房间列表：`homeData##房间1,房间2`。
- WebSocket 返回游戏数据：`gameData##payload`。
- 页面/API 端口和 WebSocket 端口是两回事。例如 `http://103.91.210.141:55555/` 是页面端口，房间数据实际走 `103.91.210.141:8888/ws`。
- 对 `103.91.210.141:8888/ws` 实测抓到在线房间 `5844`。
- 抓到的样例 payload 开头：`gameData##111,5346,6841,0,81,149,64,78.146469,2,2,0,0,13==528,4810,6885,0,0,296,127,69.862022,2,2,0,0,20==...`。
- 该样例最后列出现 `13、20、6、30、15` 这类短编号，不能直接当 `801xx` 资源 ID 使用；未找到可靠映射前不要硬猜图标。

## 网页字段对照

参考文件：`D:\hl\运营版网页+APP\网页前后台\index.html`

- `rw[0]`：英雄 ID。
- `rw[1]` 或 `rw[2]`：等级字段候选。
- `rw[3]`：大招 CD，合法范围按网页逻辑约 `0..600`。
- `rw[4]`：召唤师技能 CD 主字段，合法范围 `0..180`。
- `rw[5]`、`rw[6]`：地图坐标 x/y。
- `rw[7]`：血量或死亡状态。网页逻辑中 `>100 && <=300` 表示死亡倒计时，`<=0` 表示死亡，`1..100` 表示血量百分比。
- `rw[8]`：阵营。
- `rw[9]..rw[12]`：召唤师技能 CD/扩展字段候选。
- 网页对召唤师技能 ID 主要识别 `80001..81999`；本项目 APK 资源还包含 `55551..55555` 元流之子技能图标。

## 已做的 APK 修复

- `RadarParser`：增加 `###ST:13位时间戳` 剥离，避免 payload 尾部时间戳污染字段。
- `RadarParser`：召唤师技能 ID 解析支持 `80001..81999` 和 `55551..55555`。
- `RadarParser`：英雄生命状态按网页逻辑处理，`rw[7] > 100 && <= 300` 视为死亡倒计时，`<=0` 视为死亡。
- `RadarData.Hero`：增加 `deathCd` 字段，用于保存死亡倒计时状态。
- `RadarView`：死亡或血量为 0 的英雄不再保留上一帧缓存，地图头像立即消失，避免旧 APK “先变红、隔几秒才消失”。
- `RadarView`：技能倒计时只显示 `>0`，结束后不再显示 `0`。
- `RadarView`：`显示0秒CD` 开关已接通，开启后技能面板可以显示 `0` 秒倒计时。
- `RadarView`：野怪倒计时过滤常见“未死亡/就绪”值，例如 `0、60、70、90、120、240`，避免野怪没死时一直显示 `70`。
- `HeroIconCache`：英雄头像、召唤师图标、大招图标优先加载 APK 内置资源，减少网络下载延迟导致的头像/图标闪烁或缺失。
- `HeroIconCache`：召唤师技能未上报 ID 时使用中性问号占位，不再用闪电图标误导为真实技能。
- `NativeOverlayService`：悬浮窗调节面板做了卡通蓝白风格、美化和尺寸压缩；注意不是直接使用哆啦梦版权图，而是蓝白口袋/卡通风格原创绘制。
- `NativeOverlayService`：悬浮窗调节面板已精简为 `房间 / 绘制 / 面板`，移除无效占位功能。
- `NativeOverlayService`：`一键适配` 已改为截屏识别真实小地图位置和大小，失败时才按屏幕估算。
- `MainActivity`：主页选择房间后点击 `连接房间 xxx` 自动弹出悬浮窗；未授权时先跳授权页，返回后继续连接。

## 注意事项

- Gradle 输出里可能出现 `aapt2.exe: Failed to stat file ... Illegal byte sequence`，这是中文路径显示警告；只要最终有 `BUILD SUCCESSFUL`，APK 仍然生成成功。
- 用户工作区有大量未提交改动，不要随意 `git reset`、`git checkout --` 或回滚非本轮文件。
- 如果继续排查召唤师短编号，先找可靠来源或反编译参考 APK，不要猜 `13/20/6/30/15` 对应哪个技能。
- 参考 APK：`D:\hl\运营版网页+APP\参考apk\103.91.210.141.apk`，但它有加固/保护痕迹，普通 zip 只能看到混淆资源，未直接找到映射表。
- 如果用户说“不要后台”，打包参数应使用 `-PAPP_LOGIN_MODE=frontend`。
- 如果用户说“需要后台”，打包参数应使用 `-PAPP_LOGIN_MODE=backend`。
- 页面端口如 `55555/85/80` 不等于房间数据端口；房间数据默认仍是 `8888/ws`。
